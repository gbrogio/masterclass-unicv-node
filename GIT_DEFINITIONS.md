# Git Definitions

Conhecimento sobre tipos de branches. Tipos de commits. Criações de Pull Requests.

## Tipos de branches

### Main

Branch principal do projeto. Branch de produção.
Deve ser protegida e não pode ser deletada.

### Development

Branch de desenvolvimento. Resolver conflitos de merge. Testar o sistema.

### Feature / Bugfix / Hotfix

feature: nova funcionalidade.
bugfix: correção de bug.
hotfix: correção de bug urgente.

feature/nome-da-funcionalidade
bugfix/nome-do-bug
hotfix/nome-do-hotfix

exemplo: feature/modelando-exits-table
bugfix/resolvendo-erros-no-seed-exits-table

## Tipos de commits

### feat

Adiciona uma nova funcionalidade.

### fix

Corrige um bug.

### refactor

Refatora o código. Melhora a legibilidade do código. Melhora a performance do código.

### chore

Alterações de configuração. Alterações de build.

### docs

Alterações/Criações de documentações.

### test

Adiciona ou corrige testes.
